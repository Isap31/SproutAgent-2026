# SproutAgent Roadmap

SproutAgent is being built as a focused consulting workflow prototype. The fictional Star Wars-inspired setting makes it safe and fun to practice real consulting problems without using private client data.

## V1 Objective

Build one powerful workflow:

```text
Client Intake -> Prompt Builder -> Gemini API -> Consulting Analysis -> Report Generator -> Saved Output
```

## Phase 1: Foundation

- Define Orbital Horizons Consulting
- Create planetary client profiles
- Add mock projects and risk categories
- Write documentation and architecture notes
- Build the basic repository structure

## Phase 2: V1 Report Generator

- Load mock client data from JSON
- Select a client scenario
- Build a structured prompt
- Send the prompt to Gemini
- Save the generated report as Markdown

## Phase 3: Better Structured Outputs

- Add separate report types
- Add risk-specific reports
- Add timeline-specific reports
- Improve formatting and report consistency

## Phase 4: Operational Intelligence

- Add basic scoring logic for risk level
- Add cost allocation assumptions
- Add project timeline templates
- Add success metric recommendations

## Phase 5: User Interface

Possible UI options:

- Command-line interface
- Streamlit dashboard
- Simple web app
- Future React interface

## Phase 6: Multi-Agent Expansion

Future specialized agents may include:

- Risk Agent
- Timeline Agent
- Cost Agent
- Proposal Agent
- Executive Summary Agent

## Long-Term Vision

The long-term goal is to explore how AI can support consulting work by turning messy intake information into clearer, faster, and more repeatable first-draft deliverables.
